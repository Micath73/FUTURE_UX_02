# 📱 Future Interns | UI/UX Task 2: Mobile Appointment Booking App

## 📌 Project Overview
An ultra-modern, high-fidelity Mobile Appointment Booking platform designed inside a fully interactive mobile viewport frame. This prototype features a functional dynamic filtering system and a state-management engine that simulates an automated backend scheduling service.

---

## ⚙️ Implemented Features & User Flows
1. **Premium Onboarding Screen:** High-impact splash page introducing users to the platform with a responsive call-to-action trigger.
2. **Dynamic 20-Specialist Directory:** A fully populated directory spread across 5 distinct professional categories (Tech, Legal, Design, Finance, Health). 
3. **Interactive Filter System:** Selecting category chips instantly sorts the directory layout to show relevant certified specialists in real-time.
4. **State-Driven Date & Time Picker:** Captures user selection data (specific specialist profile, date card, and active morning/afternoon time slots).
5. **Mock Backend Confirmation Ticket:** Synthesizes the captured state data to print a custom success confirmation voucher featuring an automated unique tracking code (`#BK-XXXX`).

---

## 🎨 Design Theme & Technical Stack
- **Color Profile:** Trustworthy Deep Navy Blues, structured Slate Grays, and bright Accent Emerald Greens for active states.
- **Frontend Architecture:** Built with high-performance utility-first Tailwind CSS classes and custom vanilla JavaScript state-handling logic.
- **Asset Management:** Powered by optimized, universally accessible high-resolution CDN media strings.
